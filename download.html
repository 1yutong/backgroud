锁屏时间壁纸应用

自动更新锁屏壁纸，实时显示当前时间和日期

📱 应用下载说明

本项目是完整的 Android 应用，需要先构建 APK 文件才能安装到手机。

快速获取 APK 的方法：

  • 在本地使用 Android Studio 构建（推荐）
  • 使用 GitHub Actions 自动构建
  • 使用 CI/CD 平台（如 Bitrise）
📖 查看构建指南 🔗 访问 GitHub 仓库

应用特性：

自动定时更新锁屏壁纸（每分钟）
实时显示当前时间和日期
美观的深色渐变背景
前台服务确保持续运行
开机自启动功能
支持 Android 7.0+ 系统

🚀 快速构建命令

如果你已安装 Android Studio：

1. 打开 Android Studio
2. 导入 LockScreenWallpaper 项目
3. 点击 Build → Build APK
4. 在 app/build/outputs/apk/debug/ 找到 APK

版本：1.0.0 | 大小：约 5MB | 支持 Android 7.0+
